# W8 Quiz: Count one to three
Write a program that can print the number from 1 to 3.

## Requirements
1. You should use recursion to implement the program.
2. You should implement two directions of the program:
    - From 1 to 3
    - From 3 to 1

## Expected Output
- From 1 to 3:
```
1
2
3
```
- From 3 to 1:
```
3
2
1
```
