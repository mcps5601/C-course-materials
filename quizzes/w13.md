# Find the first index of a substring in a string

## Description:
Write a program to find the first index of a substring in a string. If the substring is not found, return -1.

## Example1
- Inputs
```
programming
gram
```
- Outputs
```
3
```
## Example2
- Inputs
```
hello
world
```
- Outputs
```
Substring not found.
```
